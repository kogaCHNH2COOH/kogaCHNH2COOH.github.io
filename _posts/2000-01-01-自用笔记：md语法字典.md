表格演示一下
| 公式 | 效果 | 代码`$$替换$$` |
| :--: | :--: | :-----------: |
| 数学字母 | $x$ | `$x$` |
| 简单公式 | $x+y=1$ | `$x=y=1$` |
| 分数 | $\frac{1}{2}$ | `\frac{1}{2}` |
| 带字母的分数 | $\frac{x+1}{y+10}$ | `\frac{x+1}{y+10}` |
| 平方（上标） | $x^2$ | `x^2` |
| 下标 | $x_2$ | `x_2` |
| 上下标同时 | $x_2^2$ | `x_2^2` |
| 上下标多数 | $x_2^{2x}$ | `x_2{2x}` |
| 根号 | $\sqrt{4}$ | `\sqrt{4}` |
| 多次开根 | $\sqrt[3]{4}$ | `\sqrt[3]{4}` |
| 中括号 | $\sqrt{[(4+x)-3]}$ | `\sqrt{[(4+x)-3]}` |
| 大括号需转译 | 我编不出来了总之是`\{替换\}` | `\sqrt{\{[(4+x)-3]\}}` |
